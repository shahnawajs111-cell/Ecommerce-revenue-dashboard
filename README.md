## 📌 Project Overview
This project is an interactive **E-commerce Revenue Dashboard** built using **Microsoft Power BI** to analyze sales performance, customer behavior, product categories, discounts, and profitability. The dashboard provides valuable business insights that help stakeholders make data-driven decisions.

## 📈 Insights

- Electronics generated the highest revenue.
- Standard membership customers contributed the most sales.
- Female customers generated slightly higher revenue than male customers.
- Most revenue was generated during non-holiday periods.
- Summer season recorded the highest sales.
- Wednesday produced the highest daily revenue.
- High-value customers contributed over 70% of total revenue.
